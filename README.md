# To-Do App (Task Management Web Application)

A feature-rich **To-Do Web Application** built using **HTML, CSS, and Vanilla JavaScript** to efficiently manage daily tasks with priorities, deadlines, reminders, and persistent storage.

---

## Features

- Add tasks with **priority levels** (Low / Medium / High)
- Set **due dates, due times**, and **reminders**
- Get browser-based reminder alerts before task deadlines
- Mark tasks as **completed / uncompleted**
- **Inline task editing** for quick updates
- Filter tasks by:
  - All / Completed / Uncompleted  
  - Priority (Low, Medium, High)
- **Soft delete** tasks with Trash & Restore functionality
- **Persistent storage** using browser `localStorage`
- Fully **responsive UI** for desktop and mobile devices
- Automatic detection of **overdue (delayed) tasks**

---

## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6)
- **State Management:** Browser LocalStorage
- **Tools:** VS Code, Git, GitHub
